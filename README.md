#  Insightify - Data Analytics Dashboard

##  Description
Insightify is an interactive data analytics dashboard that allows users to upload CSV files and visualize data using charts and statistics.

##  Features
- Upload CSV file
- Data preview
- Summary statistics
- Line chart and bar chart visualization
- Pie chart option

##  Technologies Used
- Python
- Pandas
- Streamlit
- Matplotlib

##  How to Run

1. Clone the repository:
   git clone https://github.com/your-username/Data-Analyst-Dashboard

2. Navigate to the project folder:
   cd Data-Analyst-Dashboard

3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   streamlit run app.py
